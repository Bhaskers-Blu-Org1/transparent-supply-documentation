<PageDescription>

The Connector API rejects the XML submission and returns a 500 Internal Server Error response.

</PageDescription>

## Instructions to fix

This is an unexpected Connector API exception and will need to be investigated and
corrected by your support team. Please wait for one hour before attempting to resubmit
XML to allow time for the issue to be addressed. If the problem persists, please contact
support for further instructions.
