<PageDescription>

The Connector API rejects the XML submission and returns a 503 Service Unavailable
message.

</PageDescription>

## Instructions to fix

This exception is returned when an external resource required by the Connector API
is not accessible. Please wait for one hour before attempting to resubmit XML to
allow time for the external resource issue be corrected. If the problem persists
please contact support for further instructions.
