description: XML submission error message 403 Forbidden

<PageDescription>

The Connector API rejects the XML submission and returns a 403 Forbidden response.

</PageDescription>

## Instructions to fix

This occurs when your organization does not have the proper permissions set to
complete the operation. Please contact support if you should have permissions to
perform the operation.
