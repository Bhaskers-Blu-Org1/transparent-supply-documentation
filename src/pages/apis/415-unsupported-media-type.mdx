---
title: Connector API 415 message
description: XML submission error message 415 Unsupported Media Type
---

<PageDescription>

The Connector API rejects the XML submission and returns a 415 Unsupported Media
Type response.

</PageDescription>

## Instructions to fix

The Connector API only accepts XML and all requests require the header Content-Type with
the value application/xml.  Please ensure this request header is set, contains the
correct value, then resubmit the XML.
