<PageDescription>

**<SolutionName />** includes defined static roles that grant authorization to
human users, by registered <a target="_blank" href="https://www.ibm.com/account/us-en/signup/register.html">IBMid</a>,
to execute certain network tasks on behalf of their **organization**. Assigning
roles enables an **Account admin** to easily control the level of access provided to
each user in their organization.

</PageDescription>


## Prerequisites

When onboarding new users to <SolutionName />, an **Account admin** must assign
one or more roles to each user. Before assigning roles, the following prerequisites
must be met:

* Your **organization** and your **Account admin** **must be onboarded** to <SolutionName />.
* Each human user being onboarded must have a registered <a target="_blank" href="https://www.ibm.com/account/us-en/signup/register.html">IBMid</a>.

## Add or modify role assignments

An **Account admin** can add or modify user role assignments:

1. Log in to <a href={process.env.GATSBY_UI_URL + '/'} target="_blank"><SolutionName /></a> as an **Account admin**.
1. Select the <a href={process.env.GATSBY_UI_URL + '/manageUsers'} target="_blank">**Users**</a> module.
1. Select the **ACCOUNT USERS** tab.
1. To add roles for a new human user, select the new user icon. Add the user and select the roles to assign.
1. To modify roles assigned to a human user, select a user and click the edit icon. Select the roles to assign.
